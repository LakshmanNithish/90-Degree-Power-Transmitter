# 90° Mechanical Power Transmission

A SolidWorks CAD project demonstrating a simple mechanism for transmitting rotary motion through a **90-degree angle** using interconnected links and rotating shafts.

## Project Overview

This project was created as part of my learning in CAD design and mechanical systems using SolidWorks. The objective was to design and assemble a compact mechanism capable of transmitting rotational motion between two shafts positioned at **90°** to each other.

The complete assembly was modeled, constrained, and animated in SolidWorks to verify the motion.

## Working Principle

The mechanism transfers rotational motion from the input shaft to the output shaft positioned at **90 degrees**.

1. The input shaft rotates.
2. The connecting mechanism transfers the motion.
3. The output shaft rotates at a right angle to the input.

This demonstrates the basic concept of right-angle mechanical power transmission.
